# 🚀 Quick Start Guide - Pawfect Pet Supplies POS

## How to Use the System

### 🛍️ Making a Sale

1. **Browse Products**
   - Select category from horizontal tabs (Dog Food, Cat Toys, etc.)
   - Or search by product name/SKU in the search bar
   - Products with "Popular" tag are high-volume items

2. **Add to Cart**
   - Click on product card to add 1 item
   - Use the green + button on card
   - Or scan barcode using FAB (floating button bottom-right)

3. **Manage Cart**
   - Switch to "Cart" tab
   - Use +/- buttons to adjust quantities
   - Tap trash icon to remove items
   - See real-time total updates

4. **Member Login** (Optional)
   - Click user icon in header
   - Demo accounts:
     - Maria Santos (09171234567) - Buddy's Mom, 450 pts
     - Juan Dela Cruz (09181234567) - Whiskers' Dad, 320 pts
   - Get 5% discount on all purchases
   - Earn loyalty points (1 pt per ₱10)

5. **Checkout**
   - Click "Proceed to Checkout" from cart
   - Review order summary
   - Select payment method:
     - 💵 Cash
     - 💳 Card
     - 📱 E-Wallet (GCash/Maya)
   - Click "Complete Payment"
   - See success message
   - Cart clears automatically

## 🔍 Barcode Scanner Demo

### Using the Scanner

1. **Open Scanner**
   - Click green FAB with scan icon (bottom-right)
   - Full-screen scanner appears

2. **Automatic Scan**
   - Scanner simulates scan after 2 seconds
   - Shows green flash on success
   - Item automatically added to cart

3. **Manual Entry**
   - Click "Enter SKU Manually"
   - Type product code (e.g., DF001, CF001)
   - Click "Add to Cart"

4. **Close Scanner**
   - Tap X in top-right corner
   - Returns to products screen

### Sample SKUs to Try
```
DF001 - Premium Dry Dog Food 15kg (₱89.99)
```

## 📊 Viewing History

1. Click **History** tab (clock icon)
2. See all completed transactions
3. Each transaction shows:
   - Transaction ID
   - Date & time
   - Payment method (with colored icon)
   - Line items
   - Total amount
   - Status (Completed)
